# Potential Denitrification Rate (PDR) Manual Tool

A user-driven tool for estimating **Potential Denitrification Rate (PDR)** based on manually provided environmental and soil parameters.

---
Link:https://bbp-e-tool.share.connect.posit.cloud

## 📌 Overview

The **PDR Manual Tool** is designed to calculate denitrification potential using user-supplied data instead of automated sensors or live datasets.

This tool is especially useful for:
- Field researchers  
- Laboratory experiments  
- Educational purposes  
- Areas with limited real-time data access  

---

## 🎯 Objectives

- Provide a simple method for estimating PDR  
- Enable manual input of environmental variables  
- Support research and academic analysis  
- Allow flexible use without IoT or live data  

---

## 🚀 Features

- ✍️ Manual input of soil and environmental parameters  
- 🧮 Instant PDR calculation  
- 📊 Simple and clear output results  
- 🧪 Suitable for lab and field data entry  
- ⚡ Lightweight and easy to use  

---

## 🛠️ Tech Stack

- **Language:** R   
- **Interface:** Shiny (R) / CLI / Simple Web UI  
- **Libraries:** Basic statistical and data processing packages
